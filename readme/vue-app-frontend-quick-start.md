# Vue App Frontend Quick Start

## Install dependencies
```shell
nvm use 14.19.1
npm install
```
## Configure Auth0
```shell
cp config.development.json config.json
```

## Run the dev server
```shell
nvm use 14.19.1
npm run hot
```

## Run tests
```shell
nvm use 14.19.1
npm test
```

## Linting and formatting
```shell
nvm use 14.19.1
npm lint-fix
```
